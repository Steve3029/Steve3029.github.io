Reflection of Udacity Lesson 1

What are HTML, CSS, and DOM?

The HTML, CSS, and DOM are essential technologies for building web pages. HTML is like your home structure. It decide where is the bedroom, where the living room, and where the bathroom. The CSS is the decorations of your rooms, like the color of the wall, the style of the curtain, and so on. The DOM is like a kind of framework used to connect the rooms as a whole.

What is meant by boxifying design?

Every element on the web page is a box. The boxifying design means that we should think of the box as a basic tool and element of thinking and designing page conent and structure.

What is the box model?

The box model is a one of the most important concept in web developing and design. Every individual element of a web page is included in a box that consist of content, padding, border, and margin. The width of the element is not just its contents, but also the padding and border width of the box where it resides. And the margin will decide the distance to the next element.
